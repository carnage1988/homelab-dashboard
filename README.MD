# Homelab Health Dashboard

A personal DevOps learning project designed to monitor the health and status of my homelab infrastructure.

The project will initially monitor my ZimaBoard, with the long-term goal of migrating my services to a Mac mini and maintaining visibility of the infrastructure throughout the migration.

## Project Goals

- Monitor host CPU, memory and disk usage.
- Monitor Docker containers and services.
- Track service availability and uptime.
- Create dashboards for infrastructure health.
- Add automated health checks and alerting.
- Use Git and CI/CD practices to manage the project.
- Document the migration of services from the ZimaBoard to a Mac mini.

## Planned Technologies
